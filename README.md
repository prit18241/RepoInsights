# GitHub Repo Analyzer

A web application that analyzes GitHub repositories and provides useful insights such as repository statistics, language distribution, contributor information, and project health metrics.

## Features

- Analyze any public GitHub repository
- View stars, forks, and issues
- Repository health score
- Contributor insights
- Language statistics

## Tech Stack

- FastAPI
- Python
- HTML/CSS/JavaScript

## Installation

1. Clone the repository
2. Install dependencies
3. Run the FastAPI server
